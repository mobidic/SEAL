About

Welcome to SEAL db (Simple, Efficient And Lite database for NGS), a new source for Genomics Analysis. We're dedicated to giving you the very best of Genomics Analysis, with a focus on variant interpretation, user friendly interface, easy to deploy and maintain.

Released in 2020 by MoBiDiC, SEAL has come a long way from its beginnings in 2020. When MoBiDiC first started out, their passion to improving the genetic diagnosis drove them to developped a lot of tools so that SEAL can offer you the best analysis experience.

We hope you enjoy our system as much as we enjoy offering it to you. If you have any questions or comments, please don't hesitate to contact us.

SEAL is developed and maintained by the MoBiDiC group of the Montpellier University Hospital.

v1.0.0 [January 5, 2023]

We are proud to announce the first official release of SEAL, a web application for analyzing and annotating genetic variants.

In this initial release, SEAL includes the following key features:

  • An intuitive interface for uploading and managing VCF files
  • Annotation of variants using the Ensembl Variant Effect Predictor (VEP)
  • Sample and family management tools for tracking variant inheritance patterns
  • A rich set of visualizations and reports to help interpret variant data
